##  Project Overview
This project is an advanced Machine Learning system designed specifically for the Daffodil International University (DIU) grading scale. It predicts a student's **Final Letter Grade (A+ to F)** based strictly on their continuous pre-final assessment marks (Out of 60). This allows educators to identify at-risk students *before* the 40-mark Semester Final Examination and provide timely intervention.

##  Advanced Features
1. **DIU Custom Data Simulation:** Generates a highly realistic dataset of 1,000 students, strictly following the DIU Assessment Task marks (Attendance: 7, Class Test: 15, Assignment: 5, Presentation: 8, Midterm: 25).
2. **Multi-Class Classification:** Uses an optimized Random Forest Classifier with Hyperparameter Tuning (via `GridSearchCV`) to predict 10 distinct letter grades rather than a simple binary Pass/Fail.
3. **Interactive UI (Streamlit):** A modern, multi-page web application featuring:
   - **Single Grade Predictor:** Interactive sliders to test individual student scenarios.
   - **Bulk Class Predictor:** Allows teachers to upload a CSV of an entire roster and instantly download AI-predicted grades for the whole class.
   - **Analytics Dashboard:** Displays dynamic insights including Feature Importance, Correlation Heatmaps, and the Model's Confusion Matrix.
